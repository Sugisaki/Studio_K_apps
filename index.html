気候カレンダー（二十四節気七十二候）

二十四節気、七十二候、雑節とその説明をカレンダーから表示するアプリです。俳句の歳時記としてもご利用頂けます。

Round Run 1.1

Round Run.（ラウンドラン） は、走行時間や距離、走行履歴、歩数などの詳細な情報を表示します。地図上で走行軌跡を表示し、丁目レベルで地名が変わった際には音声で地名を読み上げ ます。また、GPXファイルも出力できるので、配送の記録や車の運行記録にも使えます。タイマーを使えば、インターバルウォーキングも快適に行えます。 さらに、ライトモードとダークモードの使い分けも可能です。Round Run で、より詳細な走行記録が残せます。

天命カレンダー（四柱推命、天中殺）

使い方は、設定画面で生年月日を設定して、カレンダーを見るだけ。 カレンダーでは天中殺（空亡）の日にマークが表示されています。 算命学、四柱推命は中国の陰陽五行説を元にして人の命運を推察する方法です。 流行りの占いとは一線を画す統計によって導かれた伝統的な方法です。 派手さはありませんが、生活のリズムをこの運勢の周期に少しだけ合わせることで快適に過ごせるかもしれません。 ※ 天中殺（空亡）の日は、積極的な変化への行動は控え、勉強や休息や老人と過ごすなどの生活が推奨されています。
