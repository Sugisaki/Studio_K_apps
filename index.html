気候カレンダー（二十四節気七十二候）

二十四節気、七十二候、雑節とその説明をカレンダーから表示するアプリです。俳句の歳時記としてもご利用頂けます。

Round Run 1.1

走行時間や距離、走行履歴、歩数などの詳細な情報を表示します。地図上で走行軌跡を表示し、地名が変わった際には音声で読み上げます。

天命カレンダー（四柱推命、天中殺）

生年月日を設定するだけで天中殺（空亡）の日がわかるカレンダーです。算命学や四柱推命に基づいた生活のリズム作りに役立ちます。

このサイトについて

サイトの目的

当サイト（Studio K Applications）は、日々の生活や特定の趣味・活動をより豊かに、そして便利にするための独自のWebアプリケーションやAndroidアプリを開発・公開しています。技術的な探求心から生まれたツールを、広く一般の方々にもご利用いただくことで、少しでも皆様のお役に立つことを目的としています。

機能の概要

現在、ランニングやサイクリング、登山などで記録されたGPSの軌跡（GPXファイル）を地図上に表示し、標高や速度のグラフを分析できる「Round Run GPXビューア」を公開しています。また、日本の伝統的な暦である二十四節気・七十二候を手軽に確認できるAndroidアプリや、日々の運勢のリズムを知るためのカレンダーアプリなども提供しており、それぞれが特定のニーズに応えるユニークな機能を持っています。

プライバシーとセキュリティ

当サイトで提供するツール、特に「Round Run GPXビューア」は、ユーザーのプライバシーとセキュリティを最優先に設計されています。お客様がビューアで読み込むGPXファイルは、お使いのブラウザ内でのみ処理が完結し、外部のサーバーにアップロードされたり、保存されたりすることは一切ありません。あなたの個人的な活動記録が、意図せず外部に共有される心配はありませんので、安心してご利用ください。サイト全体の信頼性を担保するため、プライバシーポリシーも各ツールページで明記しています。
